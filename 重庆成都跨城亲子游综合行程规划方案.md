#### 景点详情

**重庆动物园**（重庆特色景点）
- 地址：重庆市九龙坡区杨家坪西郊一村1号
- 门票：25元/人，1.2米以下儿童免费
- 开放时间：8:00-17:00（16:30停止入园）
- 预约：无需预约，现场购票
- 特色亮点：
  - 全国大熊猫数量最多的城市动物园（现共有23只）
  - 可同时观赏成年大熊猫和幼崽
  - 儿童动物互动区可触摸羊驼
  - 两栖爬行馆展示巨型陆龟
- 亲子建议：
  - 早9点熊猫最活跃，建议南门进入
  - 10:30-11:00为熊猫投喂时间
  - 带防晒帽和驱蚊贴，园内推车友好

**山城步道**（重庆特色景点）
- 地址：重庆市渝中区山城巷
- 门票：免费
- 开放时间：全天
- 特色亮点：
  - 保留原汁原味的老重庆风貌
  - 青石板路、吊脚楼、老巷子
  - 沿途可俯瞰长江美景
  - 隐藏名人故居、传统小吃
- 亲子建议：
  - 步道有坡度，注意安全
  - 可选择平缓路段游览
  - 沿途有休息点和小吃店

**白象居**（重庆特色景点）
- 地址：重庆市渝中区解放东路和白象街交汇处
- 门票：免费
- 开放时间：全天
- 特色亮点：
  - 24层高楼无电梯，山城"8D魔幻"建筑代表
  - 1楼、10楼、15楼都有出入口，每个出口都是马路
  - 可拍摄长江索道、东水门大桥等地标
  - 港风复古电影感拍照圣地
- 亲子建议：
  - 楼层较高，注意看护孩子
  - 选择低楼层游览即可

- 地址：重庆市渝中区嘉陵江滨江路
- 门票：免费
- 开放时间：全天
- 特色亮点：
  - 国内首个为轨道交通奇观量身打造的观景空间

  - 重庆特色交通工具，连接上下城区
  - 全程运行约2分30秒
- 亲子建议：
  - 扶梯较长，注意安全
  - 可往返体验

- 门票：免费
- 开放时间：全天
- 特色亮点：
  - 成都最繁华商业街
  - IFS爬墙熊猫打卡点
  - 太古里古建筑与现代融合
  - 美食购物一站式
- 亲子建议：
  - 可在IFS看爬墙熊猫
  - 太古里适合拍照散步
  - 周边美食丰富

## 三、成都段亲子友好型景点详解

### 1. 成都大熊猫繁育研究基地

| 项目 | 详情 |
|------|------|
| 地址 | 成都市成华区熊猫大道1375号 |
| 门票 | 成人55元，儿童（6周岁以下或1.3米以下）免费 |
| 开放时间 | 夏季7:30-18:00，冬季8:00-17:30 |
| 预约方式 | 官方公众号"成都熊猫基地"预约购票 |
| 建议游览时间 | 3-4小时 |
| 最佳游览时段 | 8:00-11:00（熊猫最活跃） |

**游玩路线推荐：**
```
南门入口 → 观光车站 → 大熊猫1号别墅 → 大熊猫2号别墅 
→ 大熊猫产房 → 小熊猫区 → 大熊猫博物馆 → 出园
```

**特色体验项目：**
- 大熊猫产房观看幼崽
- 小熊猫散养区互动
- 大熊猫博物馆科普
- 4D影院观影（另收费）

**儿童互动环节：**
- 大熊猫知识问答
- 科普讲解服务
- 熊猫主题纪念品商店

**安全提示：**
- 园区内禁止投喂动物
- 不要使用闪光灯拍照
- 保持安静，不要大声喧哗
- 看护好儿童，不要翻越护栏

## 七、亲子出行注意事项

### 1. 儿童健康防护

#### 出行前准备
- [ ] 携带儿童医保卡/身份证

- 高铁/飞机上注意儿童耳压调节（可准备糖果）

- 选择正规餐厅就餐
- 告知商家"宝宝辣度"或"不辣"
- 避免生冷食物
- 自备儿童餐具更卫生
